The whole keyboard as ONE Faust program: 12 master oscillators feed phase-locked octave dividers, and all 48 note channels - each with its own CD4007 gate envelope and KORG35 cell - run every sample with no voice allocation, exactly like the hardware. Keys just set the gate bitmask. The KBD trigger conditioning (SINGLE holds through legato; MULTIPLE re-fires per attack) gates the shared GEG → VCA, then resonators and ensemble. Octaves don't beat - they're the same divider chain. Hold a fistful of keys.
